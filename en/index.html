What is vcpkg?
Vcpkg is an open sourced MIT License C++ package and library manager. We have stable support for Windows, Mac OS, Linux and community support for iOS, Android, and other architectures. We designed vcpkg to work with CMake, MSBuild, Make, and other buildsystems.

Why vcpkg?

Flexible

We have everything you might expect in a package manager tool (versioning, manifests) and more (enterprise CI/CD features, binary caching).

Reliable

We designed vcpkg to be easy to use and easy to integrate with any existing build environment.

Secure

Vcpkg tries to take build times to the next level through all sorts of caching, parallelism, and other methods.

Open

Vcpkg aims to provide consistent and stable builds across developer and CI/CD workflows.
